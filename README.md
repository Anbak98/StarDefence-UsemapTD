### 맵 디자인
- 타일맵 사용
    - 타일 스프라이트 솔팅 순서가 이상하 것은, 스프라이트 아틀라스로 묶어서 해결 [참고](https://docs.unity3d.com/kr/2021.3/Manual/Tilemap-Isometric-RenderModes.html)

### 핵심 기능
[필수] 전투 루프: 적 생성 및 이동, 영웅 공격, 적 체력 감소, 적 사망처리 (보스 x)
[필수] 웨이브: 승리/실패 판정 (지휘관 체력 기반)
[필수] 영웅 시스템: 소환 및 승급 (재화소모 포함)

### 추가 기능
[선택] 영웅 초월, 현상금, 탐사정, 강화, 수리, 타일 버프

### 기타 개선 요구사항
1. "Stage" 타일 사이즈가 다른 타일들과 다릅니다.
2. 배경 스프라이트가 분리되지 않아 동적 해상도 대응이 힘듭니다.
    - 카메라 폭을 배경 스프라이트 폭에 맞췄습니다. 해상도에 따라 스테이지가 잘릴 가능성이 있습니다.
